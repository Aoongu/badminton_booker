🏸 抢场神器 · Fiddler 注入方案

利用你已有的 Fiddler，把抢场面板注入到系统页面 · 完全无跨域

💡 原理 Fiddler 代理微信的网络请求 → 拦截系统首页的 HTML 响应 → 在里面追加我们的脚本 → 脚本在 bdtyg.cugb.edu.cn 自己的域名下运行 → 发请求不受任何跨域限制
⚙️ 第一步：在 Fiddler 里修改脚本规则
1
打开 Fiddler，点菜单栏 Rules → Customize Rules…
会打开一个 FiddlerScript 编辑器（.js 文件）
2
找到 OnBeforeResponse 函数
用 Ctrl+F 搜索 OnBeforeResponse，找到这个函数体
3
在函数体最开头追加下面这段代码
复制后粘贴到 static function OnBeforeResponse(oSession: Session) { 的下一行
// ★ 羽毛球抢场注入 ★ if (oSession.HostnameIs("bdtyg.cugb.edu.cn") && oSession.uriContains("/") && oSession.oResponse.headers.ExistsAndContains("Content-Type","text/html")) { oSession.utilDecodeResponse(); var body = oSession.GetResponseBodyAsString(); if (body.IndexOf("_bk_inject") == -1) { var script = "<script src='https://cdnjs.cloudflare.com/ajax/libs/crypto-js/4.1.1/crypto-js.min.js'></script><script>" + getBkScript() + "</script>"; body = body.Replace("</body>", script + "</body>"); oSession.utilSetResponseBody(body); } }
4
在同一个文件末尾（最后一个 } 前）追加辅助函数
这是抢场面板的完整代码，复制后粘贴到文件末尾
见下方「复制完整注入脚本」按钮
5
保存文件（Ctrl+S），Fiddler 自动编译生效
右下角状态栏显示 Script updated 表示成功
📱 第二步：使用抢场面板
1
确保 Fiddler 在运行，微信走 Fiddler 代理
就像你平时抓包时一样开着 Proxifier + Fiddler 即可
2
在微信里打开预约系统，刷新页面
右下角会出现一个 🏸 蓝色浮动按钮
3
点击按钮 → 配置 → 启动定时
选好日期（+1/+2/+3天）、场地、时间段，点「启动」，关闭手机睡觉
7:30 到点自动发请求，你只需要起来滑验证码 + 输密码
⚠️ 注意 每次抢完场后，可以在 Fiddler 里把注入代码注释掉（加 //），避免每次打开系统都显示面板
